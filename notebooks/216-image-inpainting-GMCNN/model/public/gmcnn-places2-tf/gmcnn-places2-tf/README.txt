This is the TensorFlow implementation of the GMCNN model trained on the Places2
dataset from the following repository:

    https://github.com/shepnerd/inpainting_gmcnn

converted to the TensorFlow frozen graph format for use by Open Model Zoo
(https://github.com/openvinotoolkit/open_model_zoo).
